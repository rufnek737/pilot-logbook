Pilot Logbook SVG Assets

- pilot_logbook_app_icon.svg: app icon design wrapped as SVG, 1024x1024.
- pilot_logbook_splash_screen_portrait.svg: splash screen, 1080x1920.
- pilot_logbook_splash_screen_panel.svg: square-ish splash panel from the original design.
- pilot_logbook_splash_phone_mockup.svg: phone mockup preview, not recommended for actual app splash.
- pilot_logbook_splash_vector_simple.svg: clean vector-only splash fallback. This is not the exact generated photo design, but works when a pure SVG/vector file is required.

Note: The exact design SVGs preserve the generated image by embedding PNG data inside the SVG. They are visually identical, but not fully editable vector paths.
