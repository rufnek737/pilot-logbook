PILOT LOGBOOK

개인정보처리방침

시행일: 2026년 7월 20일

Pilot Logbook은 서비스 제공에 필요한 최소한의 정보만 처리합니다. CrewConnex 아이디와 비밀번호 및 파싱을 위해 선택한 PDF·Excel·CSV 원본 파일은 Pilot Logbook 서비스 코드에 영구 저장하지 않습니다. 기존 경력 증빙 첨부파일은 사용자의 기기에만 저장되고 서버로 전송되지 않습니다.

3. 처리 방법

  • 비행기록은 기기 로컬 저장소와 로그인 계정의 Firebase Firestore에 저장됩니다.
  • PDF·Excel·CSV 파일은 사용자 기기에서 분석하며, 분석된 비행기록만 동기화 대상이 됩니다.
  • 기존 경력 증빙 PDF·사진은 계정별로 구분된 기기 내부 저장소에만 보관되며 Firebase 또는 Pilot Logbook 서버에 업로드하지 않습니다.
  • 전체 백업 파일은 사용자의 명시적인 요청으로 생성되어 사용자가 선택한 파일 앱·iCloud Drive·Google Drive 등에 전달됩니다. 해당 외부 보관 위치의 이용조건과 보안은 각 서비스 및 사용자의 관리에 따릅니다.
  • CrewConnex 로그인 정보는 HTTPS로 Cloudflare Worker에 전송되어 사용자가 요청한 조회에만 사용됩니다. Pilot Logbook 코드는 해당 아이디와 비밀번호를 데이터베이스·파일·캐시에 영구 저장하지 않습니다.

5. 제3자 제공

Pilot Logbook은 사용자의 개인정보를 판매하지 않습니다. 법령에 근거가 있거나 사용자가 별도로 동의한 경우를 제외하고, 서비스 제공 목적과 무관한 제3자에게 개인정보를 제공하지 않습니다. CrewConnex 자동 가져오기는 사용자가 입력한 계정으로 해당 제3자 서비스에 접속하는 기능이며 CrewConnex 자체 정책의 적용을 받을 수 있습니다.

6. 파기 절차 및 방법

사용자는 앱의 ‘계정 및 도움말 → 회원 탈퇴’에서 계정 삭제를 시작할 수 있습니다. 본인 확인 후 Firebase 로그인 계정, Firestore 비행기록, 현재 기기의 로컬 비행기록과 해당 계정의 기기 내 경력 증빙 첨부파일을 삭제합니다. 삭제한 기록은 복구할 수 없습니다. 사용자가 파일 앱이나 외부 클라우드에 별도로 보관한 백업 파일은 Pilot Logbook이 삭제할 수 없으므로 사용자가 직접 삭제해야 합니다.

7. 이용자의 권리

이용자는 자신의 비행기록을 조회·수정·삭제할 수 있고, 회원 탈퇴를 통해 계정과 관련 정보의 삭제를 요청할 수 있습니다. 개인정보 열람·정정·삭제·처리정지 또는 기타 문의는 아래 이메일로 접수할 수 있습니다.

8. 안전성 확보 조치

  • 인증과 클라우드 데이터 접근은 Firebase 사용자 계정을 기준으로 분리합니다.
  • 외부 통신에는 HTTPS를 사용합니다.
  • 로그아웃 시 기기의 계정별 로컬 비행기록을 제거해 다른 계정과 섞이지 않도록 합니다.
  • 기기 전용 경력 증빙은 로그인 계정 식별값으로 분리해 다른 계정 화면과 백업에 노출되지 않도록 합니다.
  • CrewConnex 비밀번호를 Pilot Logbook 데이터베이스에 저장하지 않습니다.

9. 방침 변경

처리 항목, 외부 서비스 또는 유료 기능이 변경되면 시행 전에 앱 또는 공개 페이지에서 변경 내용을 안내합니다. 중요한 변경은 합리적인 기간을 두고 알립니다.
